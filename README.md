## 🚛 Telematics Fleet Management Device (MQTT-Based)

This project is a lightweight telematics device designed for real-time fleet monitoring and management using the MQTT protocol. It enables vehicles to securely transmit key data such as GPS location, speed, engine status, and diagnostics to a centralized platform with low latency and minimal bandwidth usage.

Built for scalability and reliability, the device leverages MQTT’s publish/subscribe architecture to ensure efficient communication between vehicles and backend systems. It is suitable for logistics, transportation, and fleet operators looking to gain better visibility, improve operational efficiency, and enhance vehicle safety.

### Key Features

* 📡 Real-time GPS tracking
* 🔄 MQTT-based data transmission (GPS,Live fleet Map,Wifi,RFID Scan, Heartbeat)
* 🔐 Secure and scalable communication
* ⚙️ Easy integration with cloud platforms and dashboards
